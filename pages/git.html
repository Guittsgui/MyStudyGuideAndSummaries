Configurando SSH/Github

Uma das primeiras coisas que precisarmos fazer para trabalhar com o gitHub é estabelecermos uma conexão entre a nossa máquina e nossa conta do github. O jeito mais seguro de fazer isso é configurando através das Chaves SSH. Para isso o PRIMEIRO PASSO é gerarmos a chave através do comando: ssh-keygen -t ed25519 -C "your_github_email@example.com".


O SEGUNDO PASSO é ativarmos nosso SSH AGENT através do comando: eval "$(ssh-agent -S)".


O TERCEIROPASSO após termos gerado nossa chave, precisamos copia-la, para abrir e copiar, basta utilizarmos o comando: cat /home/joaozinho/.ssh/id_ed25519.pub depois irmos até o nosso github => settings => adicionar nova chave ssh.


Para confirmar se a conexao foi estabelecida, ssh -T git@github.com.
